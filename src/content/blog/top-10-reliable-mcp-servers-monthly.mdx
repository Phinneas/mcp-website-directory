Every month, we rank MCP servers by a **composite reliability score (0–100)** built from five purely data-driven metrics — no user-submitted reviews, no editorial picks, no sponsorship. Just verifiable signals from GitHub and package registries.

## Scoring Methodology

| Dimension | Weight | Source |
|-----------|--------|--------|
| Stars trajectory | 30% | GitHub star count + recent commit activity |
| Issue response time | 25% | Median days to close issues (last 90 days) |
| Fork activity | 20% | Total forks + recent fork recency |
| Download trend | 15% | npm weekly downloads + week-over-week growth |
| Commit frequency | 10% | Average commits per week (last 90 days) |

Minimum threshold: 40/100. Servers below this are excluded.
