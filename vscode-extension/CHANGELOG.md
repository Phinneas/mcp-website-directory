# Changelog

All notable changes to the My MCP Shelf extension.

## [0.1.0]

### Added
- Initial release.
- **Browse Curated Servers** command — ranked, trust-scored servers in the palette.
- **Search in Plain English** command — natural-language search via the website's shared engine, with per-result "why" reasons.
- **Browse by Category** command.
- **One-click install** to Cursor (`~/.cursor/mcp.json`), Claude Desktop (`claude_desktop_config.json`), VS Code workspace (`.vscode/mcp.json`), clipboard, or website — generating config identical to the website's Config Generator.
- Security/trust/adoption scoring from the bundled directory snapshot (28 manually audited servers).
- `myMcpShelf.liveData` setting for fresh-score enrichment from mymcpshelf.com.
